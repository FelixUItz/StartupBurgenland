## Image Guidelines

1. **Format**: Use JPG for photos, PNG for logos with transparency, SVG for vector logos
2. **Quality**: High resolution images (at least 2x the display size for retina)
3. **Optimization**: Compress images for web (use tools like TinyPNG or ImageOptim)
4. **Naming**: Use lowercase, hyphen-separated filenames
5. **Location**: Place all images in `/images/` folder
